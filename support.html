Support

There are several different ways to get help for Notpod. All your options are listed below, and any of them lets you get in touch with the application developers to get your questions answered.

We also have a straight forward user guide available, which shows you the basic steps of installing and setting up Notpod to synchronize your device. Please have a look at this user guide to see whether it answers your question. If it does not; please get in touch with us and we'll be happy to help!

Notpod Help
We use Google Groups as our support forum for Notpod. Visit the Notpod Help group to post your question!
Notpod Dev
Are you playing around with the Notpod source code and have questions to the development team? Use the Notpod Dev group to get in touch with us!
Report bugs
In order to improve the application and user experience, we rely on feedback from the users and community. If you have found a bug, or have a suggestion for improvement, please report it in our issue tracker.
